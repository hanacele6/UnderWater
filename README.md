# UnderWater
潜水艦に閉じ込められた主人公たちが脱出を目指すSF推理ゲーム
現在は作成途中です。Unityで開けばいくつかの機能を体験できますが、肝心のゲームフロウ実装には至っていません。

主なあらすじ
主人公たちはある日突然潜水艦の中で目を覚ます。
年齢性別はバラバラ、残された手掛かりはこの潜水艦が何らかの実験に使われていたということと、その目標のみ。
主人公たちは潜水艦を操作し、手に入れた有機物からサンプルを培養、潜水艦本来の目的を達成し、脱出を目指す。

～一応ネタバレ注意～

主人公たちは潜水艦を探索していく中で気が付く。
この潜水艦の乗組員の名前が、自分たちと一致していることに。

そして、彼らがすでに死んでいることに。

約10年前
かつてこの潜水艦ではとある生命体の研究が行われていた。
その生命はどんな生物のどんな器官の細胞も再現できるという夢のような生物だった。
もちろん強力な生体兵器も作れるだろう。
しかしその生物は意思を持っており、機械的に利用することができなかった。
そこで日本は優秀な科学者、自衛隊を集め、海底で極秘に研究を行うことにした。

生命体は人間との接触で急激に知性を増していく。いつしかコミュニケーションが可能なほどに。
生命体たちは思った。どうしたら人間が喜ぶのかと。
そして実行した。人間たちが求めていたとあるものを作ることにした。

動かなくなった人間を彼らは分析し、再現した。それも、それぞれの人間の最も印象に残っていた時の姿を。
